# 올라가냥: 구름잠 (Cloud Nap) — v2

귀여운 고양이가 구름과 지붕을 타고 위로 오르는 코지 플로우 게임.

- **스와이프로 발판 선택**: 매 박자, 머리 위로 떠오른 2개의 발판(나무/보상/구름) 중 하나를 좌우로 스와이프(또는 탭=더 좋은 쪽)해서 고른다. 하단 버튼 없음 — 화면 전체가 스와이프 영역.
- **졸음(체력)으로 압박**: 시간 게이지 대신 '졸음'. 깔끔한 착지·콤보·보상으로 깨어있음을 채우고, 늘어지면 졸린다. 고양이 눈과 작은 달 게이지로 표시.
- **즉사 없음**: 잘못 골라도 귀여운 제자리 비틀거림(기분 1칸↓). 기분이 0이 되면 그 자리에서 잠들며 한 판 종료 — 떨어져 죽지 않는다.
- **미끄러운 구름**: 구름 발판은 다음 박자에 옆으로 미끄러질 수 있다(콤보 8+면 면역).
- 집 → 지붕 → 하늘 → 노을 → 우주로 펼쳐지는 배경, 곳곳의 친구냥.

순수 정적 웹게임(바닐라 JS + Canvas). 모든 아트는 오리지널 코지 카툰 스타일.
